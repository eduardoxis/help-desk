# help-desk
O Helpdesk Back-End é uma aplicação desenvolvida em Java com Spring Boot, projetada para gerenciar atendimentos, chamados técnicos e suporte ao usuário. Seu objetivo é fornecer uma estrutura confiável para que empresas ou equipes de TI possam registrar problemas, acompanhar solicitações e controlar todo o fluxo de suporte de forma organizada.
